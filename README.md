# dvd-rental-sql-analysis

## Project Overview
This project analyzes customer rental behavior and revenue trends using SQL.

## Dataset
The dataset includes film, rental, inventory, customer, and payment data.

## Key insights
- Identified top customers contributing to revenue
- Analyzed the most rented and highest revenue-generating films
- Discovered seasonal trends in rental activity

## Tools Used
- PostgreSQL (pgAdmin 4)
- SQL
- Excel

## Project Files
- SQL queries
- Analysis report (includes visualizations)
